# THERIS-An-IoT-Based-Thermal-Imaging-Robot-and-Alert-System-for-Search-and-Rescue-Operations-
A Search and Rescue Drone integrated with IoT (local network) and a web-based user interface using a Raspberry Pi, Infiray P2 Pro thermal camera, microwave motion sensor, GPS, and Pixhawk flight controller
